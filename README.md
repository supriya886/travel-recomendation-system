# Travel Recommendation System

## 📌 Description
This project is a Travel Recommendation System that suggests destinations based on user preferences such as budget, interests, and location.

## 💻 Technologies Used
- HTML
- CSS
- JavaScript

## ⚙️ Features
- User-friendly interface
- Destination suggestions
- Interactive design
- Responsive layout

## 🚀 How to Run
1. Download the project
2. Open interface.html in browser
